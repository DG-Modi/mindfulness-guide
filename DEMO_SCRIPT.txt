[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let us look at how it works behind the scenes. When you enter a query, it is first validated by our Security Checkpoint node, which scrubs personal information and filters prompt injection attacks. If it is safe, the workflow passes control to the Orchestrator Node, which acts as the main hub. From there, the Orchestrator delegates tasks to two specialized sub-agents: the Wellness Coach and the Mood and Screen Time Tracker. These sub-agents retrieve data and execute actions using our local Model Context Protocol or MCP server. Finally, before starting any routine, the graph halts at the Human Confirmation Node to pause for your approval. (pause)
