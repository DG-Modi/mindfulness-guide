[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Let's look at the diagram to see how it works. When you type a message, it first goes to the Security Checkpoint. This checks for safety, hides personal info like emails, and blocks bad prompts. If it is safe, the message goes to the Orchestrator Node, which is the main brain. The Orchestrator sends the task to the right sub-agent: either the Wellness Coach or the Mood and Screen Time Tracker. These sub-agents run tools through our local MCP server to fetch exercises or log data. Finally, before starting any routine, the app stops at the Human Confirmation Node to ask for your permission.
